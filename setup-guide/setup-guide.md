> 這份手冊會帶你裝好 **Claude 桌面版（Claude Desktop）**，並用**公司寄給你的組織邀請**完成登入。上課用的就是這個 App 裡的 **Code** 功能。

> 卡住了不用緊張 —— 課程有三場助教日專門陪大家解問題。

## 全部只有 5 步

1. 接受公司的組織邀請信
2. 下載 Claude 桌面版
3. 安裝到電腦
4. 登入（用邀請信那個帳號）

做完第 5 步看到 Claude 幫你建好一個檔案，就代表你準備好了 ✅

## 第 1 步：接受公司的組織邀請信

1. 到你的 **hexschool 公司信箱（@hexschool.com）** 收信，打開 **Anthropic 組織邀請信**（寄件人是 Anthropic，主旨是邀請你加入組織）
2. 點信中的 **Accept invite（接受邀請）** 按鈕
3. 瀏覽器會打開：
   - **還沒有 Claude 帳號** → 它會請你用**收到邀請的這個 email** 設定密碼、完成註冊
   - **已經有帳號** → 直接用那個帳號登入即可
4. 完成後，你的帳號就會被加進公司的組織裡

〔截圖：組織邀請信 → Accept invite 畫面〕

> ✅ **成功的樣子**：你看到 Claude 的網頁畫面，帳號已屬於公司組織。
>
> ⚠️ **務必用收到邀請信的 hexschool 信箱帳號**。用別的信箱（個人 Gmail 等）會找不到組織、上課當天用不了。

---

## 第 2 步：下載 Claude 桌面版（Claude Desktop）

1. 前往 👉 **https://claude.com/download**
2. 找到 **「Desktop」** 這一塊（標題是 Desktop、說明寫「All of Claude, in one app」）
3. 在它底下點 **macOS** 那一行的 **Download**
4. 等待 `.dmg` 檔下載完成（通常在「下載項目」資料夾）

〔截圖：claude.com/download 的 Desktop 區塊，框出 macOS 的 Download〕

> ⚠️ **認明「Desktop」這塊就好。** 頁面上若還看到「Claude Code」「Enterprise deployment」或要你開終端機貼指令的安裝法，**那些都不是我們要的**，直接跳過。我們要的就是 Desktop 的 **macOS** 版。
>
> 💡 不確定自己的 Mac 是 Intel 還是 Apple Silicon 也沒關係，這個版本兩種都能裝。

---

## 第 3 步：安裝到電腦

1. 點兩下剛剛下載的 **`.dmg`** 檔打開它
2. 會跳出一個視窗，把 **Claude 圖示**用滑鼠**拖到 Applications（應用程式）資料夾**
3. 到「應用程式」資料夾，點兩下 **Claude** 打開

〔截圖：DMG 拖曳到 Applications 的畫面〕

> ⚠️ **如果 Mac 跳出「無法打開，因為來自未識別的開發者」**：
> 到 **系統設定 → 隱私權與安全性**，往下找到 Claude 那一行，點 **「仍要打開」**，再回去開一次就好。

> ✅ **成功的樣子**：Claude 桌面版視窗打開了。

---

## 第 4 步：登入

1. App 打開後會請你登入，點 **Sign in**
2. 用**第 1 步接受邀請的那個帳號**登入
3. 若它跳到瀏覽器要你確認登入，點確認後**回到 App**（必要時關掉 App 重開一次）
4. **如果登入後出現「選擇組織」** → 選 **hexschool**（不要選你的個人空間）

> ✅ **成功的樣子**：App 上方中央看得到 **Chat / Cowork / Code** 三個分頁。

---

## 第 5 步：開「Code」分頁，做一次成功驗收

這步是最重要的確認 —— 確定你上課當天真的能動手。

1. 點上方中央的 **Code** 分頁
   - 👉 如果它叫你 **Upgrade（升級）** 或說沒權限，代表**組織還沒生效或選錯帳號**：確認第 1 步邀請已接受、第 4 步登入／選的是 **hexschool** 組織；還是不行就私訊洧杰。
2. 環境選 **Local**（用你自己電腦上的檔案）
3. 點 **Select folder（選擇資料夾）**
   - 為了測試，先在**桌面**新建一個空資料夾，命名 `claude-test`，選它
4. 在下方輸入框打這句話，然後按 Enter：

   > 在這個資料夾建立一個 hello.txt，裡面寫一句歡迎我來上 AI 工作術體驗營的話

5. Claude 會提出「要建立 hello.txt」，畫面出現 **Accept（接受）／Reject（拒絕）**，點 **Accept**

〔截圖：Code 分頁 + Accept/Reject 的 diff 畫面〕

> ✅✅ **大成功的樣子**：你打開桌面的 `claude-test` 資料夾，裡面真的多了一個 **hello.txt**，內容是 Claude 寫的歡迎詞。
>
> 看到這個檔案 = 你已經完全準備好了，課堂上就是用這個方式請 Claude 幫你做事 🎉

## 常見問題 FAQ

**Q：這要花錢嗎？要自己訂閱嗎？**
A：**不用**。這次用的是公司的 Anthropic 組織帳號，費用公司統一付，你只要接受邀請信就能用，不用自己刷卡、不用報帳。

**Q：我沒收到組織邀請信？**
A：先檢查 hexschool 信箱的垃圾信件匣，找寄件人是 **Anthropic**、主旨含 invite 的信。還是沒有就私訊洧杰，確認有沒有把你的 email 加進組織。

**Q：我點 Code 分頁，它一直叫我升級（Upgrade）或說沒權限？**
A：通常是①邀請還沒接受，或②登入／選到的是個人帳號不是 hexschool 組織。回第 1、4 步確認；都對還不行就私訊洧杰，可能 seat 還沒指派。

**Q：我要先會寫程式嗎？要裝 Node.js、Git 那些嗎？**
A：**基本上不用**——你不用會寫程式，全程不碰終端機。唯一的例外是最後一關「寫程式腳本」：AI 會幫你寫一支小程式並當場執行，那需要電腦先裝好 Node.js。到 [nodejs.org](https://nodejs.org/) 下載 LTS 版、雙擊裝到底即可（Mac 是 .pkg、Windows 是 .msi），其他練習完全用不到。

**Q：Mac 說「無法打開、來自未識別的開發者」？**
A：到「系統設定 → 隱私權與安全性」，找到 Claude 點「仍要打開」即可（見第 3 步）。

**Q：登入後叫我選組織、或找不到 Code 分頁？**
A：登入時請選 **hexschool**（不是個人空間）。選錯會看不到組織的方案，Code 分頁會要你升級。

**Q：我是 Windows 電腦怎麼辦？**
A：這份手冊以 Mac 為主。若你用 Windows，請先私訊洧杰，我們提供 Windows 版步驟（下載頁的 Desktop 區塊同樣有 Windows 版）。

**Q：卡住了怎麼辦？**
A：別硬撐。① 對照本手冊「成功的樣子」逐步確認卡在哪一步；② **直接私訊洧杰**回報；③ 或帶到第一場助教日，我們現場陪你解。
